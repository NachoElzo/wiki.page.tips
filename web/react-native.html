  • Brew
  • Install Watchman for preventing watch files error in execution

Installation

  • Create a folder where your react-native projects will be stored
  • Open a terminal inside you project
  • Create project with expo, type in terminal:  npx create-expo-app@latest
  • Options with blank project and project name:  npx create-expo-app@latest --template blank projectName

Npm scrips for run in differents devices

  • npm start
    • you will have to select the options displayed for run in differents devices
  • npm run android
  • npm run ios
  • npm run web

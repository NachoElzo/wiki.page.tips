Linux shell commands for jenkins

  • list all files inside the directory  sh 'ls'
  • list all files and hidden files inside the directory  sh 'ls -la'
  • Return an string  sh 'echo string chars'
  • Creates a folder  sh 'mkdir folderName'
  • Creates a folder if the folder do no esxist  sh 'mkdir -p folderName'
  • Creates a file inside the folder  sh 'touch folderName/fileName.txt'
  • Insert txt into the file  sh 'echo "Text to insert" >> folderName/fileName.txt'
  • Open and reads the file in console  sh 'cat folderName/fileName.txt'
  • verify if the file exists  sh 'test -f folderName/fileName.txt'
  • Remove a file  sh 'rm folderName/fileName.txt'
  • Search for a string inside a file  sh 'grep "textToFind" folderName/fileName.txt'
  • Search for more than one string inside a file  sh 'grep "textToFind" folderName/fileName.txt' && sh 'grep "other text" folderName/fileName.txt'
    * [sh] Before the command is to advice jenkins that we are using shell commands
